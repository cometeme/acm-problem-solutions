# 1207F. Remainder Problem

> time limit: 4s | memory limit: 512mb

You are given an array $a$ consisting of $500000$ integers (numbered from $1$ to $500000$). Initially all elements of $a$ are zero.

You have to process two types of queries to this array:

-   $1$ $ğ‘¥$ $ğ‘¦$ â€” increase $a_x$ by $y$ ;

-   $2$ $ğ‘¥$ $ğ‘¦$ â€” compute $\sum _{i \in R(x,y)} a_i$, where $R(x,y)$ is the set of all integers from $1$ to $500000$ which have remainder $y$ modulo $x$. 

Can you process all the queries?

##### Input

The first line contains one integer $q$ ($1 \leq q \leq 500000$) â€” the number of queries.

Then $q$ lines follow, each describing a query. The $i$-th line contains three integers $t_i$, $x_i$ and $y_i$ ($1 \leq t_i \leq 2$). If $t_i = 1$, then it is a query of the first type, $1 \leq x_i \leq 500000$, and $âˆ’1000 \leq y_i \leq 1000$. If $t_i=2$, then it it a query of the second type, $1 \leq x_i \leq 500000$, and $0 \leq y_i < x_i$ .

It is guaranteed that there will be at least one query of type $2$.

##### Output

For each query of type $2$ print one integer â€” the answer to it.

##### Example

Input
```text
5
1 3 4
2 3 0
2 4 3
1 4 -4
2 1 0
```
Output
```text
4
4
0
```
